getAffineTransform
==================
.. image:: http://kube.pl/wp-content/uploads/2018/04/getAffineTransform_1.png

Functionality
-------------
Calculates an affine transform from three pairs of the corresponding points.


Inputs
------
- pts1_in – Pts1 input.
- pts2_in – Pts2 input.


Outputs
-------
- matrix_out – Output matrix.
